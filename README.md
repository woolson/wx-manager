# wx-manager

> web app for manage wechat image and article

### Functions

- upload multi images once
- remove image
- upload article

### Structure
```bash
├── README.md
├── images    # uploaded images
├── server    # server
└── web       # pages of web
```

### Start

```bash
npm run start
# or
node server index.js
# or (need pm2 installed)
pm2 start server/index.js
```
